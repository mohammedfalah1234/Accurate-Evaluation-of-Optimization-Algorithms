Overview

This is a comprehensive, production-grade system for evaluating, comparing, and recommending optimization algorithms. Unlike traditional approaches that rely on a single metric (e.g., "best fitness value"), this system evaluates algorithms using 20 physical metrics and provides a score out of 100.

Key Numbers

Metric Value
Total Algorithms 9,860 (60 single + 9,800 hybrid)
Physical Metrics 20 (5 points each = 100 total)
Supported Engines MealPy 3.0+, NiaPy 2.0+, SciPy
Data Formats CSV, TSV, JSON, XLSX, XLS, TXT, DAT, NPY + direct paste

What Makes This System Unique?

1. 20 Physical Metrics for Algorithm Evaluation

Instead of asking "which algorithm is best?", this system answers: "which algorithm is best for your specific problem?" using:

· Convergence Speed
· Solution Quality
· Exploration-Exploitation Balance
· Escape from Local Optima
· Energy Efficiency
· Memory Footprint
· Search Space Coverage
· Uncertainty Quantification
· And 12 more physics-based metrics

2. Data Advisor (NEW in v5.0)

Upload your data (CSV, Excel, JSON, or paste directly) and the system:

· Computes 10 physical equations on your data (Ruggedness, Modality, Entropy, Epistasis, etc.)
· Runs a tournament across all 9,860 algorithms
· Recommends the optimal single and hybrid algorithm with physical explanations

3. Hybrid Algorithm Generator

Generates voting ensembles from 2 to 50 algorithms. Each hybrid:

· Assigns a specialized role to each component algorithm
· Uses a voting mechanism (chooses the best result at each iteration)
· Avoids the weaknesses of individual algorithms while combining their strengths

4. System Architect (Generative AI for Algorithms)

Creates brand new optimization algorithms tailored to your data using:

· Genetic Programming (15-gene bank)
· Auto-Evolution (8 candidates evaluated, elite crossover)
· AST + compile + runtime verification
· Automatic scientific documentation generation

5. Adversarial Stress Test

Tests algorithm robustness against 8 mathematical attacks:

· Gaussian Noise
· Extreme Value Flip
· Feature Dropout
· Outlier Injection
· Covariate Shift
· Spurious Correlation
· Sign Flip Attack
· FGSM-style Perturbation
